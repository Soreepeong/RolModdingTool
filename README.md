# SynergyTools
Tool for dealing with Sonic Boom: Rise of Lyric file formats, and maybe modding?

## Features
* Compress and extract .wiiu.stream archives.
* Repack files to play as Shadow or Metal Sonic.
* ...and more to come. Maybe.
